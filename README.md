# Personal Portfolio

A personal portfolio website built with React.

## Tech Stack

- **React** — UI framework
- **Vite** — build tool and dev server
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **Vercel** — deployment

## Features

- Animated photo collage on hero
- Favorites section with swipeable categories (movies/shows, artists, games)
- AI chat assistant (used Gemini API)
- Auto-hiding navbar
- Expandable cards in experience section

## Running Locally

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel. Every push to `main` triggers an automatic redeploy. The Gemini API key is stored securely in Vercel's environment variables.
